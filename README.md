# 家用用電與延長線過載檢查器 (Web)
一個純前端實用工具：輸入家電瓦數/數量/使用時間，立即估算電流(A)、耗電(kWh)、電費，並檢查是否可能過載。

## 功能
- 家電清單輸入（瓦數W、數量、每天小時）
- 自動計算：總功率、電流(A)、月耗電(kWh)、電費
- 安全檢查：是否可能超過插座/延長線/斷路器設定上限
- 純前端：可直接用 GitHub Pages 展示

## 上線（GitHub Pages）
1. 新建 repo
2. 上傳 README.md + index.html
3. Settings → Pages → Source 選 main / root
4. 開啟網站網址即可展示
